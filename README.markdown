# ActsAsFavorite

### General information

Adds functionality to models.

## Installation

To use it, add it to your Gemfile:

    gem 'acts_as_favorite', :git => 'git@github.com:ivoreis/acts_as_favorite.git'
    bundle update

### Post Installation

1. rails g acts_as_favorite:install
2. rake db:migrate
